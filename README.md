# CollabZen – Smart Team Management Platform

CollabZen is a modern, cloud-ready platform that simplifies the way teams manage projects, assign tasks, track progress, and collaborate.  
Now upgraded with **PostgreSQL** and fully deployed on **NeonDB** for high-performance, scalable database operations.

---

## 🌟 Features

### 🔐 Authentication & Authorization
- Secure **JWT-based login & signup**
- Role-based access control (Admin / Manager / Member)

### 📁 Project & Task Management
- Full **CRUD** for projects and tasks  
- Assign members, set deadlines & priorities  
- Upload attachments via Multer  
- Track completion and progress  

### 🔎 Searching, Sorting & Filtering
- Search by project name, task title, or assignee  
- Sort tasks by priority, date, or status  
- Filter by project, deadline, or completion  

### 📊 Analytics Dashboard
- Task completion statistics  
- Member performance charts using **Chart.js**

### 🔔 Real-Time Updates
- **Socket.io** for instant notifications and updates

### 🌐 Deployment
- **Frontend**: Vercel  
- **Backend**: Render  
- **Database**: PostgreSQL hosted on **NeonDB**
